## Quick Reference

### Settings Locations

| Setting | Location |
|---------|----------|
| API Keys | Settings → AI Provider |
| Model Selection | Settings → AI Provider |
| System Prompt | Settings → Chat |
| Temperature | Settings → Chat |
| Memory | Settings → Memory |
| Auto-Compact | Settings → Compact |
| PIN Lock | Settings → Safety |

| Command | Action |
|---------|--------|

| `/rename` | Auto-rename conversation |
