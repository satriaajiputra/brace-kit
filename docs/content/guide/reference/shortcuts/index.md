# Keyboard Shortcuts

Quick reference for all BraceKit keyboard shortcuts.

## Global Shortcuts

These work when the sidebar is open.

| Shortcut | Action |
|----------|--------|
| `Enter` | Send message |
| `Shift+Enter` | New line in input |
| `Esc` | Cancel streaming / Close popovers |

## Browser Shortcuts

These work anywhere in the browser.

| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift+B` | Toggle sidebar (configurable) |

> **Note:** You can customize browser shortcuts at `chrome://extensions/shortcuts`

## Input Area

| Shortcut | Action |
|----------|--------|
| `Enter` | Send message |
| `Shift+Enter` | Insert new line |
| `Tab` | Accept autocomplete suggestion |
| `Esc` | Cancel autocomplete |

## Message Actions

| Shortcut | Action |
|----------|--------|
| `C` | Copy message (when hovered) |
| `E` | Edit message (user messages) |
| `B` | Branch from message (assistant messages) |
| `R` | Regenerate response (assistant messages) |

## Navigation

| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift+B` | Open/close sidebar |
| `Esc` | Close settings, popovers |
| `?` | Show shortcuts (on docs pages) |

## Slash Commands

Type these in the input area:

| Command | Action |
|---------|--------|
| `/compact` | Summarize conversation |
| `/rename` | Auto-generate conversation title |
| `/` | Show command autocomplete |

## Lightbox (Images)

| Shortcut | Action |
|----------|--------|
| `←` | Previous image |
| `→` | Next image |
| `Esc` | Close lightbox |
| `F` | Toggle favorite |
| `C` | Copy image |

## History Drawer

| Shortcut | Action |
|----------|--------|
| `Enter` | Open selected conversation |
| `Delete` | Delete selected conversation |
| `Esc` | Close drawer |

## Customizing Shortcuts

### Browser Shortcuts

1. Open `chrome://extensions/shortcuts`
2. Find "BraceKit"
3. Click the pencil icon
4. Press your preferred key combination

### Available Actions

- **Toggle sidebar**: Open/close the sidebar

> **Note:** Only the sidebar toggle can be customized at the browser level.

## Tips

### Efficient Chatting

1. Type your message
2. Press `Enter` to send
3. Press `Esc` to cancel if needed
4. Use `Shift+Enter` for multi-line messages

### Quick Actions

1. Hover over a message
2. Press the action key (C, E, B, R)
3. Action executes immediately

### Navigation

1. Press `Ctrl+Shift+B` to open sidebar
2. Type your question
3. Press `Enter`
4. Press `Ctrl+Shift+B` to close
